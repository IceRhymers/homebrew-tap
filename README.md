# IceRhymers Homebrew Tap

Unofficial Homebrew tap for Databricks AI Gateway proxy tools.

> These tools are unofficial workarounds to enable Databricks OAuth SSO authentication with AI coding tools. Not supported or endorsed by Databricks.

## Install

    brew tap IceRhymers/tap
    brew install databricks-claude
    brew install databricks-codex
    brew install databricks-opencode

## Tools

- **databricks-claude** — Databricks OAuth proxy for Claude Code
- **databricks-codex** — Databricks OAuth proxy for OpenAI Codex CLI
- **databricks-opencode** — Databricks OAuth proxy for OpenCode CLI
